# MotionGame 驱动层与应用层解耦设计

> 本文档说明硬件底层驱动如何与应用层解耦，作为后续开发的参考。

---

## 一、整体架构

```
┌─────────────────────────────────────────────────────────────────┐
│                        应用层 (App Layer)                        │
│   menu_app | game_snake | game_balance | theme_app             │
│   ────────────────────────────────────────────────────────     │
│   业务逻辑：游戏规则、UI交互、得分系统                           │
└─────────────────────────────┬───────────────────────────────────┘
                              │ 读取 / 触发
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                      解耦层 (Abstraction Layer)                  │
│   ────────────────────────────────────────────────────────     │
│   input.h      → 统一输入入口                                   │
│   output.h     → 统一输出入口 (feedback事件)                   │
│   game_app.h   → 统一游戏接口                                   │
└─────────────────────────────┬───────────────────────────────────┘
                              │
┌─────────────────────────────┴───────────────────────────────────┐
│                       驱动层 (Driver Layer)                      │
│   ────────────────────────────────────────────────────────       │
│   mpu6050.cpp      → I2C 姿态传感器 raw 数据                   │
│   motion_service.cpp → 动作识别 (倾斜/晃动/翻滚)                │
│   display_driver.cpp → SPI LCD 128x160 双缓冲                  │
│   buzzer.cpp       → PWM 蜂鸣器                                 │
│   motor.cpp       → PWM 震动马达                               │
└─────────────────────────────────────────────────────────────────┘
```

## 二、输入系统 (姿态 → 应用)

### 数据流

```
MPU6050 (I2C) 
    ↓ raw数据
motion_service.cpp (每帧调用)
    ↓ 滤波 + 姿态解算 + 动作识别
IMU_me (全局结构体)
    ↓ 应用层直接读取
app层 (menu_app / game_snake)
```

### 应用层使用方式

```c
#include "motion_service.h"
#include "motion_type.h"

// 方式1: 检测动作事件 (推荐)
void GameUpdate() {
    if (IMU_me.motion == MOTION_SHAKE) {
        // 开始游戏
    }
    if (IMU_me.motion == MOTION_TILT_LEFT) {
        // 向左移动
    }
}

// 方式2: 读取原始角度 (精细控制)
void SnakeControl() {
    int8_t dir = 0;
    if (IMU_me.rel_pitch > 5) dir = 1;   // 前倾
    if (IMU_me.rel_pitch < -5) dir = -1; // 后仰
    // 控制蛇的方向
}
```

### 全局数据结构

```c
// motion_type.h
typedef struct {
    float roll, pitch;           // 角度 (deg)
    float ax, ay, az;            // 加速度 (g)
    float gx, gy, gz;             // 陀螺仪 (deg/s)
    float rel_roll, rel_pitch;   // 相对基准角度
    MotionType_t motion;         // 当前动作事件
} IMU;

extern IMU IMU_me;  // 全局单例
```

### 动作事件类型

| 事件 | 触发条件 | 用途 |
|------|---------|------|
| `MOTION_TILT_RIGHT` | 前倾 > 15° | 菜单→右 / 蛇→下 |
| `MOTION_TILT_LEFT` | 后仰 < -15° | 菜单←左 / 蛇→上 |
| `MOTION_ROLL_FRONT` | 右滚 > 15° | 蛇→右 |
| `MOTION_ROLL_BACK` | 左滚 < -15° | 蛇→左 |
| `MOTION_SHAKE` | 加速度突变 | 确认 / 开始 |

## 三、输出系统 (应用 → 反馈)

### 数据流

```
app层 (game_snake)
    ↓ 语义化事件
output_feedback.cpp
    ↓ 解析事件
motor.cpp / buzzer.cpp
    ↓ PWM驱动
马达 / 蜂鸣器
```

### 应用层使用方式

```c
#include "output_feedback.h"

// 简单触发
void OnEatFood() {
    Output_Feedback(FEEDBACK_EAT);      // 轻震 + 短音
}

void OnHitWall() {
    Output_Feedback(FEEDBACK_HIT);     // 强震 + 警告音
}

void OnGameOver() {
    Output_Feedback(FEEDBACK_GAME_OVER); // 长震 + 结束音
}

// 精细控制
void OnMenuMove() {
    Output_MotorVibrate(100, 30);       // 马达: 速度100, 30ms
    Output_BuzzerTone(800, 50, 50);    // 蜂鸣器: 800Hz, 50ms, 50%音量
}
```

### 预定义反馈事件

```c
// ===== 组合反馈 (马达+蜂鸣器) =====
FEEDBACK_EAT        // 吃到食物
FEEDBACK_HIT       // 撞墙
FEEDBACK_DIE       // 死亡
FEEDBACK_LEVEL_UP // 升级
FEEDBACK_SUCCESS   // 成功
FEEDBACK_ERROR     // 错误
FEEDBACK_GAME_OVER // 游戏结束

// ===== 操作反馈 =====
FEEDBACK_CLICK     // 点击
FEEDBACK_SELECT    // 选中
FEEDBACK_BACK      // 返回

// ===== 纯马达 =====
FEEDBACK_RUMBLE    // 引擎轰鸣
FEEDBACK_TICK      // 时钟节拍

// ===== 纯蜂鸣器 =====
FEEDBACK_BEEP      // 提示音
FEEDBACK_MENU_MOVE // 菜单移动
```

## 四、显示系统 (应用 → 画面)

### 数据流

```
app层 (game_snake)
    ↓ 绘图API
display_driver.cpp
    ↓ 图形算法 + 双缓冲
frame_buffer[]
    ↓ SPI
TFT_eSPI → LCD 128x160
```

### 应用层使用方式

```c
#include "display_driver.h"

// 1. 帧周期控制
void GameLoop() {
    Display_StartFrame();     // 开始帧
    Display_ClearBuffer(BLACK);
    
    DrawBackground();
    DrawSnake();
    DrawFood();
    DrawUI();
    
    Display_EndFrame();       // 刷新屏幕
}

// 2. 基本绘图
Display_DrawPixel(x, y, color);
Display_DrawLine(x0, y0, x1, y1, color);
Display_DrawRect(x, y, w, h, color);
Display_FillRect(x, y, w, h, color);
Display_FillCircle(x, y, radius, color);

// 3. 文本绘制
TextParams_t params = {
    .font = DISPLAY_FONT_5x7,
    .color = WHITE,
    .bg_color = BLACK,
    .size = 1,
    .h_align = TEXT_ALIGN_LEFT,
    .v_align = TEXT_ALIGN_TOP,
    .transparent = true
};
Display_DrawString(x, y, "SCORE: 100", &params);
```

### API 速查

| 类别 | 函数 | 用途 |
|------|------|------|
| **帧控制** | `StartFrame/EndFrame` | 双缓冲周期 |
| **清除** | `ClearScreen/ClearBuffer` | 清屏 |
| **像素** | `DrawPixel/GetPixel` | 像素点 |
| **线条** | `DrawLine` | 直线 |
| **图形** | `Draw/FillRect/Circle/Triangle` | 填充图形 |
| **位图** | `DrawBitmap1B` | 图标/图片 |
| **文字** | `DrawString/DrawInt` | 文本/数字 |

## 五、开发流程

### 新增游戏的模板

```c
// my_game.h
#pragma once
#include "system_state.h"

void MyGame_Init(void);
SystemState_t MyGame_Update(void);
int MyGame_GetScore(void);

// my_game.c
#include "my_game.h"
#include "display_driver.h"
#include "motion_service.h"
#include "output_feedback.h"

void MyGame_Init(void) {
    // 初始化游戏状态
    Output_Feedback(FEEDBACK_SUCCESS);  // 开机反馈
}

SystemState_t MyGame_Update(void) {
    // 1. 读取输入
    if (IMU_me.motion == MOTION_SHAKE) {
        // 开始游戏
    }
    
    // 2. 游戏逻辑
    // ...
    
    // 3. 渲染画面
    Display_StartFrame();
    // 绘制游戏内容
    Display_EndFrame();
    
    // 4. 触发反馈
    if (hit_wall) {
        Output_Feedback(FEEDBACK_HIT);
    }
    
    return SYS_NONE;
}
```

## 六、关键约定

1. **全局单例**: 输入数据通过 `IMU_me` 全局访问
2. **语义化事件**: 输出使用 `FEEDBACK_xxx` 事件而非原始API
3. **帧周期**: 显示渲染必须在 `StartFrame/EndFrame` 之间
4. **周期性调用**: `loop()` 中每帧调用:
   - `MPU6050_Update()`
   - `MotionService_Update()`
   - `OutputFeedback_Update()`
   - `GameApp_Update()` / `MenuApp_Update()`
