# MotionGame 硬件接线说明

> 本文档记录所有硬件模块的接线信息，来源：源码 `*.h` 文件中的 `#define` 定义

---

## 硬件概览

| 模块 | 型号 | 接口 | 数量 |
|------|------|------|------|
| MCU | ESP32-C3 | - | 1 |
| IMU | MPU6050 | I2C | 1 |
| 屏幕 | ST7735 | SPI | 1 |
| 蜂鸣器 | 有源蜂鸣器 | GPIO PWM | 1 |
| 马达驱动 | L298N | GPIO PWM | 1 |

---

## ESP32-C3 引脚分配表

| GPIO | 功能 | 连接的外部设备 | 说明 |
|------|------|----------------|------|
| **0** | PWM 输出 | 蜂鸣器 (BUZZER_PIN) | 蜂鸣器信号 |
| **2** | SPI SCK | 屏幕 ST7735 | SPI 时钟 |
| **3** | SPI MOSI | 屏幕 ST7735 | SPI 数据输出 |
| **4** | I2C SDA | MPU6050 | I2C 数据线 |
| **5** | I2C SCL | MPU6050 | I2C 时钟线 |
| **7** | SPI CS | 屏幕 ST7735 | 片选信号 |
| **8** | GPIO 输出 | L298N IN3 | 马达B PWM |
| **12** | GPIO 输出 | L298N IN1 | 马达A PWM |
| **13** | GPIO 输出 | L298N IN4 | 马达B 方向 |
| **18** | GPIO 输出 | L298N IN2 | 马达A 方向 |

## 详细接线

### 1. MPU6050 六轴传感器

```
ESP32-C3          MPU6050
---------         --------
GPIO 4 (SDA)  →  SDA
GPIO 5 (SCL)  →  SCL
3.3V           →  VCC
GND            →  GND
```

| MPU6050 引脚 | 连接 |
|--------------|------|
| VCC | 3.3V (ESP32) |
| GND | GND |
| SDA | GPIO 4 |
| SCL | GPIO 5 |
| AD0 | GND (地址 0x68) |

> ⚠️ I2C 地址为 0x68（AD0 接地）

---

### 2. ST7735 1.8" LCD 屏幕 (128x160)

```
ESP32-C3          ST7735
---------         -------
GPIO 2 (SCK)   →  SCK
GPIO 3 (MOSI)  →  SDA
GPIO 7 (CS)    →  CS
3.3V           →  VCC
GND            →  GND
```

> ⚠️ ST7735 工作电压为 3.3V，**不要连接 5V**

| ST7735 引脚 | 连接 | 说明 |
|-------------|------|------|
| VCC | 3.3V | 供电 |
| GND | GND | 地 |
| SCK | GPIO 2 | SPI 时钟 |
| SDA | GPIO 3 | SPI 数据 |
| CS | GPIO 7 | 片选 |
| RESET | 3.3V 或 GPIO | 复位（低电平有效） |
| AO/DC | 可不接 | 数据/命令选择 |

> 📝 SPI 初始化代码：`SPI.begin(2, -1, 3, 7)` → SCK=2, MISO=-1, MOSI=3, CS=7

---

### 3. 蜂鸣器

```
ESP32-C3          蜂鸣器
---------         --------
GPIO 0          →  信号引脚
3.3V            →  VCC (有源蜂鸣器)
GND             →  GND
```

| 蜂鸣器引脚 | 连接 |
|------------|------|
| VCC | 3.3V |
| GND | GND |
| SIG | GPIO 0 (PWM) |

> 📝 PWM 配置：频率 2000Hz，分辨率 8-bit

---

### 4. L298N 马达驱动模块

```
ESP32-C3          L298N
---------         ------
GPIO 12 (IN1)  →  IN1    (马达A PWM)
GPIO 18 (IN2)  →  IN2    (马达A 方向)
GPIO 8  (IN3)  →  IN3    (马达B PWM)
GPIO 13 (IN4)  →  IN4    (马达B 方向)
5V             →  5V (逻辑电源)
GND            →  GND
```

| L298N 引脚 | 连接 | 说明 |
|------------|------|------|
| IN1 | GPIO 12 | 马达A PWM 控制 |
| IN2 | GPIO 18 | 马达A 方向控制 |
| IN3 | GPIO 8 | 马达B PWM 控制 |
| IN4 | GPIO 13 | 马达B 方向控制 |
| 5V | ESP32 5V | 逻辑电源 |
| GND | ESP32 GND | 地 |
| 12V | 电池正 | 马达电源 (如果使用独立供电) |
| OUT1/OUT2 | 马达A | 连接震动马达 |
| OUT3/OUT4 | 马达B | 未使用 |

> ⚠️ L298N 需要独立的马达电源（建议 6-12V），不要直接从 ESP32 取电

## SPI 配置

| 参数 | 值 |
|------|-----|
| SCK | GPIO 2 |
| MOSI | GPIO 3 |
| MISO | 未使用 (-1) |
| CS | GPIO 7 |
| 频率 | 默认 |

---

## 电源需求

| 模块 | 电压 | 电流 | 来源 |
|------|------|------|------|
| ESP32-C3 | 3.3V / 5V | - | USB-C 输入 |
| MPU6050 | 3.3V | ~5mA | ESP32 3.3V |
| ST7735 | 3.3V | ~30mA | ESP32 3.3V |
| 蜂鸣器 | 3.3V / 5V | ~20mA | ESP32 3.3V/5V |
| L298N 逻辑 | 5V | ~30mA | ESP32 5V |
| L298N 马达 | 6-12V | ~1A | 外部电源 |

> ⚠️ 建议使用 USB-C 供电（5V/500mA）同时给 L298N 外接独立电源（7.4V 或 12V）给马达供电

```cpp
#define BUZZER_PIN         0
#define BUZZER_BASE_FREQ   2000
#define BUZZER_PWM_RES    8
```

```cpp
#define SCL 5
#define SDA 4
```

### display_driver.cpp (SPI 初始化)
```cpp
SPI.begin(2, -1, 3, 7);  // SCK=2, MISO=-1, MOSI=3, CS=7
```
