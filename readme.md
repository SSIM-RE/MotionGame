# MotionGame

基于 ESP32-C3 的体感游戏设备 - 求职作品 + 自媒体项目

## 软件平台

* Arduino + ESP-IDF

## 硬件
- ESP32-C3
- MPU6050 (IMU 传感器)
- LCD 128x160
- 无源蜂鸣器
- 直流震动器

## 如何烧录
- Board: ESP32C3 Dev Module
- 波特率：921600
- 端口：选择正确的 COM 端口
